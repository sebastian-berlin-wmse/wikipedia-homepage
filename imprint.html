Über dieses Angebot

wikipedia.de ist ein Suchportal für die freie Enzyklopädie Wikipedia (wikipedia.org). Es bietet einen einfachen Zugriff auf Dienste von Drittanbietern, um die Inhalte der Wikipedia zu erschließen.

Suche - Um einen Begriff zu suchen, tippen Sie diesen bitte einfach in das Suchfeld ein. Es erscheint automatisch eine Liste von passenden Artikeln in der jeweils gewählten Sprache. Wählt man einen dieser Links, gelangt man direkt zum entsprechenden Artikel. Alternativ kann der Begriff auch in den verschiedenen angebotenen Suchmaschinen nachgeschlagen werden. Über die Einstellungen können Sie bequem die von Ihnen bevorzugte Suchmaschine auswählen.

Sprache - Wikipedia ist ein multilinguales Projekt. Zum Start bieten wir Ihnen deshalb neben der Suche in der deutschsprachigen Wikipedia auch Zugriff auf einige in Deutschland gesprochene Regional- oder Minderheitensprachen. Über die Einstellungen können Sie bequem Ihre bevorzugte Sprache auswählen.

Feedback - Wir freuen uns auf Ihre Vorschläge zur Verbesserung unseres Angebotes. Dafür wurde unter https://meta.wikimedia.org/wiki/Wikipedia.de eine Wiki-Seite eingerichtet.
